# Yakuza組織 ステータス確認

現在実行中のYakuza組織のタスク進捗状況を以下の手順で表示してください：

## 実行手順

1. `TaskList`を使用して、すべてのタスク一覧を取得
2. 各タスクの状態を確認:
   - **進行中** (in_progress) - 現在実行中のタスク
   - **完了** (completed) - 完了したタスク
   - **待機中** (pending) - まだ開始していないタスク
3. 必要に応じて`TaskGet`や`TaskOutput`でタスクの詳細情報を取得
4. 以下のフォーマットで整形して表示

## 出力フォーマット

```
【Yakuza組織 タスク状況】

■ 進行中のタスク (X件)
  - [タスクID] タスク名 (担当エージェント)
  - [タスクID] タスク名 (担当エージェント)

■ 完了したタスク (X件)
  - [タスクID] タスク名
  - [タスクID] タスク名

■ 待機中のタスク (X件)
  - [タスクID] タスク名
  - [タスクID] タスク名
```

## 特別な状況

- **タスクがない場合**: 「現在、実行中のタスクはありません。若頭が待機しています。」と表示
- **エラーが発生している場合**: タスクIDと共にエラー内容を表示
- **長時間実行中のタスク**: 開始時刻も表示して状況を明確に

## 注意事項

- このコマンドは**表示のみ**で、タスクの状態を変更しません
- バックグラウンドで実行中のタスクの最新状態を確認できます
- 若頭、若頭補佐、若衆の階層構造は関係なく、すべてのタスクを表示します
