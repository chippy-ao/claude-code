# law-study

e-Gov法令APIを活用した法令勉強支援プラグイン。

## 前提条件

- [bun](https://bun.sh/) がインストールされていること

## インストール

```bash
/plugin install law-study@chippy-ao-plugins
```

## 使い方

```bash
/law 民法
```

または引数なしで対話的に開始:

```bash
/law
```

### フロー

1. キーワードを入力して法令を検索
2. 一覧から学習したい法令を選択
3. 法令の要約や質問応答で学習

### 検索式

より精度の高い検索が可能:

| 種類 | 書き方 | 例 |
|------|--------|-----|
| AND | スペース区切り | `情報 公開` |
| OR | パイプ区切り | `情報公開\|個人情報` |
| NOT | 感嘆符 | `!個人情報` |
| 組み合わせ | 丸括弧 | `(情報 公開)\|個人` |

## 外部API

本プラグインは [e-Gov法令API](https://laws.e-gov.go.jp/api/2/swagger-ui) を利用しています。
